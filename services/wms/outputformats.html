WMS output formats¶

WMS는 지원하는 몇 가지 형식으로 이미지를 반환합니다. 이 페이지에서 산출물 형식의 목록을 보입니다. 산출물 형식을 설정하는 구문(syntax)은 다음과 같습니다.

format=<format>

이때 <format> 자리에 다음 옵션들 가운데 하나를 입력합니다.

주석

WMS의 GetCapabilities 요청을 통해 GeoServer 인스턴스가 지원하는 산출물 형식 목록을 찾을 수 있습니다.

형식 구문 비고
PNG format=image/png 기본값입니다.
PNG8 format=image/png8 PNG와 동일하지만, 최적의 256색(8-bit) 색상표를 계산하기 때문에 일반적으로 이미지 용량이 더 작습니다.
JPEG format=image/jpeg  
GIF format=image/gif  
TIFF format=image/tiff  
TIFF8 format=image/tiff8 TIFF와 동일하지만, 최적의 256색(8-bit) 색상표를 계산하기 때문에 일반적으로 이미지 용량이 더 작습니다.
GeoTIFF format=image/geotiff TIFF와 동일하지만, GeoTIFF 메타데이터를 담고 있습니다.
GeoTIFF8 format=image/geotiff8 TIFF와 동일하지만, GeoTIFF 메타데이터를 담고 있으며 최적의 256색(8-bit) 색상표를 계산하기 때문에 일반적으로 이미지 용량이 더 작습니다.
SVG format=image/svg  
PDF format=application/pdf  
GeoRSS format=rss  
KML format=kml  
KMZ format=kmz  
OpenLayers format=application/openlayers OpenLayers HTML 응용 프로그램을 생성합니다.
