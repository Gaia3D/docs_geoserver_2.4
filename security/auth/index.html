Authentication¶

인증에 관련된 3개의 GeoServer 리소스가 있습니다.

  • Web Administration Interface (웹 관리자(web admin)로도 알려져 있습니다.)
  • OWS 서비스 (예시: WFS, WMS 등)
  • REST 서비스
