Disabling security¶

외부 보안 하위 시스템을 이용하고 있다면 충돌을 방지하기 위해 내장(built-in) 보안을 비활성화시켜야 할 수도 있습니다. 각 필터 사슬의 보안을 개별적으로 비활성화시킬 수 있습니다. GeoServer 인증 페이지에 보안 필터 사슬 목록이 있습니다.

경고

필터 사슬의 보안을 비활성화시키면 해당 체인과 일치하는 각 HTTP 요청에 대한 관리자 권한을 허용하게 됩니다. 예를 들면 web 사슬에 대한 보안을 비활성화시키면 Web Administration Interface에 접근하는 모든 사용자에게 관리자 접근을 부여하게 됩니다.
