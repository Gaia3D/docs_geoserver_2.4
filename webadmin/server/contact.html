Contact Information¶

WMS 서버의 역량 문서(Capabilities document) 내부에서 연락처 정보를 사용하며, 공개되어 열람할 수 있습니다. 정확한 정보로 이 양식을 완성해주십시오.

연락처 페이지

Contact Information Fields¶

입력란 설명
Contact 웹마스터 연락처 정보
Organization 웹마스터가 소속된 기관명
Position 기관 내 웹마스터의 직위
Address Type 입력된 주소의 분류: 직장, 사무실 등
Address 실제 주소: 지번 주소 혹은 새주소
City 도시명
State 도(道)명
Zip code 우편번호
Country 국가명
Telephone 전화번호
Fax 팩스 번호
Email 이메일 주소
