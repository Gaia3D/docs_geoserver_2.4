Services¶

GeoServer는 Open Geospatial Consortium(OGC)이 제정한 프로토콜을 사용해서 데이터를 서비스합니다. Web Coverage Service(WCS)는 커버리지 데이터(래스터) 요청을 지원하고, 지리 피처 데이터(벡터) 요청을 지원하며, Web Map Service(WMS)는 지리 데이터에서 생성된 이미지 요청을 지원합니다.
