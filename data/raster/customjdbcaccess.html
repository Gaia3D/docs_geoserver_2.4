Custom JDBC Access for image data¶

주석

GeoServer는 초기 상태에서 Custom JDBC Access를 지원하지 못 하기 때문에 확장 모듈(extension)을 설치해야 합니다. 설치 방법은 Image Mosaic JDBC 페이지를 참조하십시오. 해당 확장 모듈이 Custom JDBC Access를 지원하고 있습니다.

Adding a coverage based on Custom JDBC Access¶

이 확장 모듈은 이미지 데이터를 저장하기 위한 특별히 데이터베이스를 설계(layout)했거나 래스터 데이터와 관련된 특수한 데이터베이스 확장 모듈을 사용하는 사용자를 위한 것입니다.

Custom JDBC Access 지원에 대한 GeoTools 문서 를 읽어보십시오.

사용자 지정 플러그인을 개발한 다음, 클래스들을 JAR 파일로 패키지 작업한 뒤 GeoServer 설치 디렉터리의 WEB-INF/lib 디렉터리로 복사하십시오.

XML 설정 파일을 생성한 다음, Image Mosaic JDBC Tutorial 페이지의 Configuring GeoServer 섹션을 참조하십시오.
