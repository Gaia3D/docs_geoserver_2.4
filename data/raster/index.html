Working with Raster Data¶

이 섹션에서 GeoServer가 접근할 수 있는 래스터(커버리지) 데이터 소스에 대해 설명합니다.

GeoServer를 표준 설치하면 다음 데이터 포맷을 로드해서 서비스할 수 있습니다.

  • GeoTIFF
  • GTOPO30
  • WorldImage
  • ImageMosaic

GeoServer 확장 모듈(extension)로 다른 데이터 소스도 사용할 수 있습니다. 확장 모듈은 GeoServer에 기능을 추가하는 모듈로, GeoServer download page 페이지에서 다운로드할 수 있습니다.

경고

GeoServer 인스턴스 버전과 확장 모듈 버전이 일치해야 합니다.

  • ArcGrid
  • GDAL Image Formats
  • Oracle Georaster
  • Postgis Raster
  • ImagePyramid
  • Image Mosaic JDBC
  • Custom JDBC Access for image data
