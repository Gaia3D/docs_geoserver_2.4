Postgis Raster¶

주석

GeoServer는 초기 상태에서 Postgis 래스터 열(column)을 지원하지 못 하기 때문에 확장 모듈(extension)을 설치해야 합니다. 설치 방법은 Image Mosaic JDBC 페이지를 참조하십시오. 해당 확장 모듈이 Postgis 래스터를 지원하고 있습니다.

Adding an Postgis raster data store¶

Postgis 래스터 지원에 대한 GeoTools 문서를 읽어보십시오. XML 설정 파일을 생성한 다음, Image Mosaic JDBC Tutorial 페이지의 Configuring GeoServer 섹션을 참조하십시오.
