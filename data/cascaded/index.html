Working with Cascaded Services¶

이 섹션에서 GeoServer가 어떻게 외부 OGC 서비스를 대리(proxy)할 수 있는지 논의합니다. 이를 종속(cascading) 서비스라고 합니다.

GeoServer는 다음과 같은 단계적 서비스를 지원합니다.

  • External Web Feature Server
  • External Web Map Server
