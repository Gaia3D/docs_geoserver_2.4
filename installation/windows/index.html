Windows¶

윈도우에서 GeoServer를 설치하는 몇 가지 방법이 있습니다. 가장 간단한 방법은 윈도우 인스톨러(Windows installer)를 사용하는 것이지만, 운영체제 독립적인 바이너리(OS-independent binary)를 이용하여 수동으로 설치할 수도 있습니다.

주석

톰캣(Tomcat)과 같은 서블릿 컨테이너의 일부로 GeoServer를 실행하려면 Web archive (WAR) 섹션을 참조하십시오.
