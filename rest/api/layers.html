Layers¶

레이어(layer)는 생산된(published) 리소스(피처형이나 커버리지)입니다.

/layers[.<format>]¶

모든 레이어를 제어합니다.

방식 동작 상태 코드 형식 기본 형식
GET 모든 레이어를 반환합니다. 200 HTML, XML, JSON HTML
POST   405    
PUT   405    
DELETE   405    

/layers/<l>[.<format>]¶

특정 레이어를 제어합니다.

방식 동작 상태 코드 형식 기본 형식 파라미터
GET 레이어 l을 반환합니다. 200 HTML, XML, JSON HTML  
POST   405      
PUT 레이어 l을 변경합니다. 200 XML,JSON    
DELETE 레이어 l을 삭제합니다. 200     recurse

Exceptions¶

예외 상태 코드
존재하지 않는 레이어에 대한 GET 404
레이어 명칭을 변경하는 PUT 403
레이어의 리소스를 변경하는 PUT 403

Parameters¶

recurse¶

recurse 파라미터는 설정된 레이어가 참조하는 모든 스타일을 반복적으로 삭제합니다. 이 파라미터에 쓸 수 있는 값은 “true” 혹은 “false”로, 기본값은 “false”입니다.

/layers/<l>/styles[.<format>]¶

설정한 레이어의 모든 스타일을 제어합니다.

방식 동작 상태 코드 형식 기본 형식
GET 레이어 l의 모든 스타일을 반환합니다. 200 SLD, HTML, XML, JSON HTML
POST 레이어 l에 새로운 스타일을 추가합니다. 201, with Location header XML, JSON  
PUT   405    
DELETE   405
