Layer groups¶

레이어 그룹(layer group)은 WMS GetMap 요청을 통해 단일 레이어로서 접근할 수 있는 레이어와 스타일의 집합입니다. 때로 레이어 그룹을 “바탕 맵(base map)”이라고 부르기도 합니다.

/layergroups[.<format>]¶

모든 레이어 그룹을 제어합니다.

방식 동작 상태 코드 형식 기본 형식
GET 모든 레이어 그룹을 반환합니다. 200 HTML, XML, JSON HTML
POST 새로운 레이어 그룹을 추가합니다. 201, with Location header XML,JSON  
PUT   405    
DELETE   405    

/layergroups/<lg>[.<format>]¶

특정 레이어 그룹을 제어합니다.

방식 동작 상태 코드 형식 기본 형식
GET 레이어 그룹 lg를 반환합니다. 200 HTML, XML, JSON HTML
POST   405    
PUT 레이어 그룹 lg를 변경합니다. 200 XML,JSON  
DELETE 레이어 그룹 lg를 삭제합니다. 200    

Exceptions¶

예외 상태 코드
존재하지 않는 레이어 그룹에 대한 GET 404
레이어가 없는 레이어 그룹을 설정하는 POST 400
레이어 그룹의 명칭을 변경하는 PUT 403

/workspaces/<ws>/layergroups[.<format>]¶

설정된 작업공간의 모든 레이어 그룹을 제어합니다.

방식 동작 상태 코드 형식 기본 형식
GET 작업공간 ws의 모든 레이어 그룹을 반환합니다. 200 HTML, XML, JSON HTML
POST 작업공간 ws 안에 새로운 레이어 그룹을 추가합니다. 201, with Location header XML,JSON  
PUT   405    
DELETE   405    

/workspaces/<ws>/layergroups/<lg>[.<format>]¶

설정된 작업공간의 특정 레이어 그룹을 제어합니다.

방식 동작 상태 코드 형식 기본 형식
GET 작업공간 ws의 레이어 그룹 lg를 반환합니다. 200 HTML, XML, JSON HTML
POST   405    
PUT 작업공간 ws의 레이어 그룹 lg를 변경합니다. 200 XML,JSON  
DELETE 작업공간 ws의 레이어 그룹 lg를 삭제합니다. 200
