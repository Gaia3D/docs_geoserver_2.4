Fonts¶

이 옵션은 GeoServer에서 사용할 수 있는 폰트(font) 목록을 제공합니다. SLD 파일을 업로드하기 전에 SLD에서 사용한 폰트(font)가 사용 가능한지 확인하는 데 유용합니다.
