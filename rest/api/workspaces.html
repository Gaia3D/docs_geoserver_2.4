Workspaces¶

작업공간(workspace)은 데이터 저장소의 집합(grouping)입니다. 명칭공간(namespace)과 유사하게, 어떤 방식으로 연관되어 있는 데이터를 무리짓는 데 사용됩니다.

/workspaces[.<format>]¶

모든 작업공간을 제어합니다.

방식 동작 상태 코드 형식 기본 형식
GET 모든 작업공간을 나열합니다. 200 HTML, XML, JSON HTML
POST 새 작업공간을 생성합니다. 201 with Location header XML, JSON  
PUT   405    
DELETE   405    

/workspaces/<ws>[.<format>]¶

특정 작업공간을 제어합니다.

방식 동작 상태 코드 형식 기본 형식 파라미터
GET 작업공간 ws를 반환합니다. 200 HTML, XML, JSON HTML  
POST   405      
PUT 작업공간 ws를 변경합니다. 200 XML, JSON    
DELETE 작업공간 ws를 삭제합니다. 200 XML, JSON   recurse

Exceptions¶

예외(Exception) 상태 코드
존재하지 않는 작업공간에 대한 GET 404
작업공간 명칭을 변경하는 PUT 403
비어 있지 않은 작업공간에 대한 DELETE 403

Parameters¶

recurse¶

recurse 파라미터는 설정된 작업공간이 참조하는 모든 레이어(데이터 저장소, 커버리지 저장소, 피처형 등등)를 반복적으로(recursively) 삭제합니다. 이 파라미터에 쓸 수 있는 값은 “true” 혹은 “false”입니다. 기본값은 “false”입니다.

/workspaces/default[.<format>]¶

기본 작업공간을 제어합니다.

방식 동작 상태 코드 형식 기본 형식
GET 기본 작업공간을 반환합니다. 200 HTML, XML, JSON HTML
POST   405    
PUT 기본 작업공간을 설정합니다. 200 XML, JSON  
DELETE   405    

/workspaces/<ws>/settings[.<format>]¶

특정 작업공간에 대한 설정을 제어합니다.

방식 동작 상태 코드 형식 기본 형식
GET 작업공간 설정을 반환합니다. 200 HTML, XML, JSON HTML
POST   405    
PUT 작업공간 설정을 생성하거나 업데이트합니다. 200 XML, JSON  
DELETE 작업공간 설정을 삭제합니다. 200 XML, JSON
