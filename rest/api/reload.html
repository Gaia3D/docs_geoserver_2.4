Reloading configuration¶

디스크에서 GeoServer 카탈로그와 설정을 다시 읽어옵니다. 외부 도구가 디스크에 저장된(on-disk) 설정을 변경했을 경우에 쓰입니다. 이 작업을 실행하면 GeoServer가 모든 내부 캐시를 비우고 모든 데이터 저장소에 다시 접속합니다.
