Styles¶

스타일(style)은 WMS가 (피처형 혹은 커버리지) 리소스에 어떤 스타일을 적용할지 설정합니다. GeoServer에서 스타일은 SLD로 설정됩니다.

/styles[.<format>]¶

모든 스타일을 제어합니다.

방식 동작 상태 코드 형식 기본 형식 파라미터
GET 모든 스타일을 반환합니다. 200 HTML, XML, JSON HTML  
POST 새로운 스타일을 생성합니다. 201 with Location header SLD, XML, JSON 다음 주석을 참조하십시오.   name
PUT   405      
DELETE   405     purge

주석

SLD 스타일과 함께 POST나 PUT 요청을 실행할 경우, Content-type 헤더를 application/vnd.ogc.sld+xml로 설정해야 합니다.

Parameters¶

name¶

name 파라미터는 스타일의 명칭을 설정합니다. 이 옵션은 SLD 형식 스타일과 함께 POST 요청을 실행할 경우에 가장 유용합니다. SLD 자체에서 해당 명칭을 추론해낼 수는 없습니다.

purge¶

purge 파라미터는 스타일의 기본이 되는 SLD 파일을 디스크에서 삭제해야 할지 설정합니다. 이 파라미터에 쓸 수 있는 값은 “true” 혹은 “false”로, 기본값은 “false”입니다. “true”로 설정할 경우, 기본이 되는 파일을 삭제합니다.

/styles/<s>[.<format>]¶

설정된 스타일을 제어합니다.

방식 동작 상태 코드 형식 기본 형식
GET 스타일 s를 반환합니다. 200 SLD, HTML, XML, JSON HTML
POST   405    
PUT 스타일 s를 변경합니다. 200 SLD, XML, JSON, 앞 주석을 참조하십시오.  
DELETE 스타일 s를 삭제합니다. 200    

Exceptions¶

예외 상태 코드
존재하지 않는 스타일에 대한 GET 404
스타일 명칭을 변경하는 PUT 403
기존 레이어가 참조하는 스타일에 대한 DELETE 403

/workspaces/<ws>/styles[.<format>]¶

설정된 작업공간의 스타일을 제어합니다.

방식 동작 상태 코드 형식 기본 형식 파라미터
GET 작업공간 ws의 모든 스타일을 반환합니다. 200 HTML, XML, JSON HTML  
POST 작업공간 ws에 새로운 스타일을 생성합니다. 201 with Location header SLD, XML, JSON, 앞 주석을 참조하십시오.   name
PUT   405      
DELETE   405     purge

/workspaces/<ws>/styles/<s>[.<format>]¶

설정된 작업공간의 특정 스타일을 제어합니다.

방식 동작 상태 코드 형식 기본 형식
GET 작업공간 ws 내의 스타일 s를 반환합니다. 200 SLD, HTML, XML, JSON HTML
POST   405    
PUT 작업공간 ws 내의 스타일 s를 변경합니다. 200 SLD, XML, JSON 앞 주석을 참조하십시오.  
DELETE 작업공간 ws 내의 스타일 s를 삭제합니다. 200
