Resource reset¶

모든 저장소, 래스터, 스키마 캐시를 리셋합니다. GeoServer가 모든 캐시를 비우고 모든 저장소 접속을 끊고, 요청이 있을 때 필요한 저장소에 다시 접속하도록 하는 데 쓰입니다. 저장소가 관리하는, 곧 변경될 수도 있는 데이터 구조에 대한 일부 정보를 저장소 스스로 캐시하고 있는 경우 유용합니다.
