Coverages¶

coverage는 커버리지 저장소에서 나온 래스터 데이터셋입니다.

/workspaces/<ws>/coveragestores/<cs>/coverages[.<format>]¶

설정된 커버리지 저장소와 작업공간의 모든 커버리지를 제어합니다.

방식 동작 상태 코드 형식 기본 형식
GET 커버리지 저장소 cs의 모든 커버리지를 나열합니다. 200 HTML, XML, JSON HTML
POST 새로운 커버리지를 생성합니다. 201 with Location header XML, JSON  
PUT   405    
DELETE   405    

/workspaces/<ws>/coveragestores/<cs>/coverages/<c>[.<format>]¶

설정된 커버리지 저장소와 작업공간의 특정 커버리지를 제어합니다.

방식 동작 상태 코드 형식 기본 형식 파라미터
GET 커버리지 c를 반환합니다. 200 HTML, XML, JSON HTML  
POST   405      
PUT 커버리지 c를 변경합니다. 200 XML,JSON    
DELETE 커버리지 c를 삭제합니다. 200     recurse

Exceptions¶

예외 상태 코드
존재하지 않는 커버리지에 대한 GET 404
커버리지의 명칭을 변경하는 PUT 403
커버리지의 커버리지 저장소를 변경하는 PUT 403

Parameters¶

recurse¶

recurse 파라미터는 설정된 커버리지가 참조하는 모든 레이어를 반복적으로 삭제합니다. 이 파라미터에 쓸 수 있는 값은 “true” 혹은 “false”로, 기본값은 “false”입니다.

Structured coverages¶

구조적(structured) 커버리지는 보통 주로 시간, 표고, 입자(granule)에 부여된 다른 사용자 지정 수치(dimension)를 나타내기 위해 사용되는 속성과 연계된 입자로 이루어져 있습니다. 이미지 모자이크(Image mosaic)는 각 모자이크 입자가 속성과 연계되어 있는 수정 가능한(writable) 구조적 커버리지 리더(reader)의 한 예입니다. NetCDF는 파일에 담겨 있는 다중 차원 그리드(multidimensional grid)를 각각 다른 변수값과 연계된 2D 슬라이스(slice)의 집합으로 노출시킨 읽기 전용(read-only) 구조적 커버리지 리더의 한 예입니다.

다음 API는 구조적 커버리지 리더에만 적용됩니다.

/workspaces/<ws>/coveragestores/<cs>/coverages/<coverage>/index[.<format>]¶

설정된 커버리지, 그 명칭, 형식, 최대/최소 발생(min/max occurrences)과 연계된 속성 집합을 선언합니다.

방식 동작 상태 코드 형식 기본 형식 파라미터
GET 속성, 그 명칭 및 형식을 반환합니다. 200 XML, JSON XML  
POST   405      
PUT   405      
DELETE   405      

/workspaces/<ws>/coveragestores/<cs>/coverages/<coverage>/index/granules.<format>¶

각 입자와 입자의 속성값 및 도형의 전체 목록을 반환하고, 선별적으로 제거할 수 있습니다.

방식 동작 상태 코드 형식 기본 형식 파라미터
GET 입자와 그 속성의 목록을 GML(XML 사용 시)이나 GeoJSON(JSON 사용 시)으로 반환합니다. 200 XML, JSON XML offset, limit, filter
POST   405      
PUT   405      
DELETE 입자를 삭제합니다. (모두 혹은 filter 파라미터를 통해 선별된 입자) 200     filter

Parameters¶

offset¶

offset 파라미터는 GeoServer에 데이터 반환 시 첫 입자의 특정 개수를 무시하도록 합니다.

limit¶

limit 파라미터는 GeoServer에 데이터 반환 시 특정 개수의 입자를 최대한 반환하도록 합니다.

filter¶

filter 파라미터는 입자의 속성값을 바탕으로 어떤 입자를 반환할지 선별할 수 있도록 하는 CQL 필터입니다.

/workspaces/<ws>/coveragestores/<cs>/coverages/<mosaic>/index/granules/<granuleId>.<format>¶

입자 하나를 반환하고, 제거할 수 있습니다.

방식 동작 상태 코드 형식 기본 형식 파라미터
GET 입자와 그 속성의 목록을 GML(XML 사용 시)이나 GeoJSON(JSON 사용 시)으로 반환합니다. 200 XML, JSON XML  
POST   405      
PUT   405      
DELETE 입자를 삭제합니다. 200
