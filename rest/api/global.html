Global settings¶

GeoServer의 일반(global) 설정에 접근할 수 있습니다.

/settings[.<format>]¶

모든 일반 설정을 제어합니다.

방식(Method) 동작(Action) 상태 코드 형식 기본 형식
GET 모든 일반 설정을 나열합니다. 200 HTML, XML, JSON HTML
POST   405    
PUT 일반 설정을 업데이트합니다. 200 XML, JSON  
DELETE   405    

/settings/contact[.<format>]¶

일반 연락처 정보만을 제어합니다.

방식 동작 상태 코드 형식 기본 형식
GET 일반 연락처 정보를 나열합니다. 200 HTML, XML, JSON HTML
POST   405    
PUT 일반 연락처 정보를 업데이트합니다. 200 XML, JSON  
DELETE   405
