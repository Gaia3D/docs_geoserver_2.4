Creating a New Data Directory¶

새로운 데이터 디렉터리를 만드는 가장 쉬운 방법은 GeoServer 표준 설치 시 따라오는 디렉터리를 복사하는 것입니다.

GeoServer가 Standalone 모드로 실행되고 있다면 데이터 디렉터리는 <installation root>/data_dir 경로에 위치합니다.

주석

윈도우 시스템에서 표준 설치 시 <installation root> 의 경로는 C:\Program Files\GeoServer <VERSION> 입니다.

GeoServer가 서블릿 컨테이너 내부에서 Web Archive 모드로 실행되고 있을 경우, 데이터 디렉터리는 <web application root>/data 경로에 위치합니다.

데이터 디렉터리의 경로를 확인했다면, 디렉터리를 외부 위치에 복사하십시오. GeoServer 인스턴스가 새로운 데이터 디렉터리 위치를 가리키게 하려면 다음 Setting the Data Directory 섹션으로 이동하십시오.
